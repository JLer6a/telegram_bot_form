# Telegram Form Bot

## ? Описание

Этот проект представляет собой Telegram-бота для сбора обратной связи от пользователей.
Бот реализован на Java с использованием Spring Boot и взаимодействует с PostgreSQL для хранения данных.
Формирует отчет в формате `.docx` с результатами, отправляемый по команде `/report`.

---

## ? Команды бота

| Команда   | Описание                                                |
|-----------|---------------------------------------------------------|
| `/start`  | Приветствие и сброс текущего состояния                  |
| `/form`   | Запуск опроса: имя ? email ? оценка (1–10)              |
| `/report` | Генерация и отправка отчета в формате Word (DOCX)       |
| `/exit`   | Прерывание прохождения формы                            |

---

## ? Технологии

- Java 17
- Spring Boot 3.1+
- PostgreSQL
- Apache POI (создание .docx)
- TelegramBots (Java Telegram Bot API)
- Docker + Docker Compose

---

## ? Сборка и запуск с помощью Docker

| Команда                | Описание                             |
|------------------------|--------------------------------------|
| `docker-compose up -d` | Поднимаем и запускаем все контейнеры |
| `docker-compose down`  | Отключаем все контейнеры             |
